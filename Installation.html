Quick Start - Installation Process ( Recommended )

1. Download the latest Startup Store source files from Codecanyon.

2. Download Install Node.js : from https://nodejs.org ! Please install latest version of Node.js.

3. Next you need to Install GIT on your system : from https://git-scm.com/ Please install latest version of GIT.
Do not worry - Git is a free and open source distributed version control system designed to handle everything from small to very large projects with speed and efficiency.

Local Project to Online

5. How to host local DATA to Github to automate your app. Aand create your own API.

i. Open project on VS Code Editor.

ii. Then Press at same time on Windows:- (Ctrl+Shift+G) and on Mac :- (⌘+Shift+G) to open Source Control

iii. Now click on Publish to Github

iv. Now type any name for easily identify eg. Store Name, Your app name

v. *IMPORTANT* -----> Select Private to Public then hit Enter.

* ----- Well done your project is now publish on Server ----- *
