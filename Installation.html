Installation Process

Quick Start - Installation Process

1. Download the latest Design - Admin theme source files from themeforest.

2. Download Install Node.js : from https://nodejs.org ! Please install latest version of Node.js.

3. (Optional) Next you need to Install if you are a git user otherwise use your own GIT : from https://git-scm.com/ Please install latest version of GIT.
Do not worry - Git is a free and open source distributed version control system designed to handle everything from small to very large projects with speed and efficiency.

---- For Gulp Users ----

Now We Completely Support Gulp Latest Version as we Promise. GULP ---Project v.1.0.0 ---

Quick Start

A. If you've previously installed gulp globally, run npm rm --global gulp before following these instructions.

B. Install the gulp command line utility : npm install --global gulp-cli

C. Go to Project Directory where you want to run/work Then run npm init

D. Now Go to Project assets Directory then select Any folder e.g. Default,Dark and you will found a gulpfile.js file. Now Open Run npm install --save-dev gulp to Install the gulp package in your devDependencies.

E. Now Verify your gulp versions run gulp --version

Next - Installation Process

4. Extract the Design - admin dashboard to your suitable directory or folder.

- After Extract you can customize with your favorite Text Editor . VS Code Editor. recommended.
Then you can save new packages Directly with npm command. Steps to save new packages --
Please follow Folder & file Structure Below ↓
