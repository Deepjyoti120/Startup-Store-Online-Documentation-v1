Installation Process

Quick Start - Installation Process
Before start please setup Flutter environment on your mechine.
If already setup then please skip it.

1. Download the latest GetProfile source files from Codecanyon.

2. Extract Startup Store vX.X.X.zip

3. Then open GetProfile folder on VS Code Editor (Recommended) OR Android Studio.

4. Connect your emulator or your phone via USB cable or WIFI debugging

5. Then simply run and test on your Device in Real time.

6. Use Flutter HOT Relode Fetures and change something and play it.

Still stuck and confusing Let's chat with you.

Note: This Project is only Tested on Android Device only. For IOS you need to setup packages using packages Documentation.
